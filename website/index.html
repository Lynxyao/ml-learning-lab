Module 1 · Image data

Can a model translate one scientific image into another?

In this module, you explore a paired image-to-image learning problem. Your goal is to understand why a conditional GAN can learn a visual mapping from microscopy input to force-map-like output, and why the result still needs scientific validation.

Guided lab

Define the ML Problem

The scientific question is whether one imaging modality can provide enough information to estimate another measurement that is harder or slower to obtain. The ML task is not classification. It is a conditional image-to-image mapping: input image in, predicted target-like image out.

Input

Wrinkle microscopy image.

Target

Aligned force-map-like image.

Model

Pix2pix-style conditional GAN.

Claim to test

Does the generated output preserve scientifically meaningful structure?

Student prediction

If the model only learns global shape, what type of error might appear in the generated force-map-like image?

Why This Model?

A conditional GAN is useful because it combines two pressures: make the image match the paired target and make the output look like a realistic target-domain image.

The generator learns to produce a target-like image from the input microscopy image.

Inspect the Paired Data

A paired training image stores input and target domains in one file before preprocessing.

Data Questions Before Training

What is one sample?

One aligned input-target image pair.

What should not leak?

The same or nearly identical pair should not appear in both train and test.

What does the model see?

The input image only. The target image is used for training loss and evaluation.

Choose the Data Split

Recommended

Use a fixed held-out test set so your result images are comparable across runs.

Explore Saved Checkpoints

WFM GAN training is slow, so the online student version uses prepared checkpoints. Choose an epoch and inspect how the generated image changes. In the code version, students can change epochs, loss weights, and checkpoint selection locally.

Epoch

More training can improve structure, but can also overfit.

GAN loss

Adds realism pressure from the discriminator.

L1 loss

Adds pixel-level target similarity, often smoothing details.

ECG Background: What Is the Signal?

              

                An electrocardiogram records voltage changes caused by the heart's electrical activity.
                A single heartbeat usually contains a P wave, a sharp QRS complex, and a T wave. In this
                module, each model input is a short 1D window around one heartbeat.
              

              

                
InputFixed-length ECG waveform segment

                
TargetAAMI-style beat class such as N, SVEB, VEB, F, or Q

                
Model1D CNN, LSTM, or CNN-LSTM

                
QuestionCan the model generalize to unseen patient records?

              

              

                Prediction before experiment
                
If normal beats dominate the dataset, which metric might look better: accuracy or macro-F1?

              

            

            

              

                
Inspect Waveforms

                
              

              
              

                
N
Normal or bundle branch block beats. This is the dominant class.

                
SVEB
Supraventricular ectopic beats. Rare classes are easy to miss.

                
VEB
Ventricular ectopic beats. Clinically important errors may be hidden by accuracy.

              

            

          

        


        

          

            

              
Data Audit Before Training

              

                The dataset is made from beat-level ECG segments. Before choosing a model, students must
                check whether the split tests true generalization, whether rare arrhythmias are represented,
                and whether the label distribution makes accuracy misleading.
              

              

                
                
              

              

                Record-wise split is harder because test records are unseen during training, but it is a better generalization test.
              

              

                
Leakage risk
Random beat splits may place similar beats from the same record in both train and test.

                
Class imbalance
Normal beats dominate, so rare arrhythmias need class-level evaluation.

                
Clinical question
The model should be judged by failure patterns, not only a single score.

Module 4 - Resistive sensor array

        

          

            
Can ML reconstruct a local resistance map from limited electrical measurements?

            

              Students explore a 3x3 resistive sensor array. The learning goal is inverse modeling:
              using measurable current signals under fixed voltage to reconstruct hidden local
              resistance values that may later correspond to cell coverage or local growth patterns.
            

          

          
Inverse sensing module

        


        

          
          
          
          
          
          
          
        


        

          

            

              
Learning Question

              

                In a cell-monitoring resistive array, each local region may have a different resistance.
                Directly and continuously measuring every internal point may be difficult. A more practical
                setup measures electrical responses such as current under fixed voltage. The ML problem is
                to infer the hidden 3x3 resistance pattern from those measurable current signals.
              

              

                Before you train
                
If two different resistance maps create similar current measurements, what extra measurement or validation data would help separate them?

              

            

            

              
Why This Is Different

              

                

                  Forward problem
                  
Given all local resistances, compute the measurable current response of the circuit.

                

                

                  Inverse problem
                  
Given current measurements, reconstruct the hidden local resistance map.

                

                

                  Research link
                  
Could fewer electrical measurements support real-time monitoring of cell growth or local abnormal regions?

                

              

            

          

        


        

          

            

              

                
Training Data Structure

                

                  
                

              

              

              
Each cell is one hidden local resistance value. The model does not see this map at test time; it only receives current measurements.

            

            

              
Input and Output

              

                
                  
Input X9 current measurements from the resistive array under fixed voltage.

                  
Target y9 resistance values reshaped into a 3x3 local resistance map.

                  
Model taskRegression: predict concrete resistance values, then evaluate high/low localization.

                  
Why this mattersThe reconstruction can indicate where the array has changed, which may later connect to local cell growth or abnormal coverage.

                
              

              

                Teaching boundary
                
The current prototype uses provided CSV data. Biological interpretation would still require experimental validation against real cell-culture measurements or imaging.

              

            

          

        


        

          

            

              
Choose Modeling Idea

              

                
                
                
              

            

            

              
Training Setup

              

                
Best current setting
MLP regression with conductance-based target transformation.

                
Physics link
Under fixed voltage, current is more directly related to conductance, or 1/R, than to raw resistance R.

Mini-Report Template

            
    
              
  • Dataset: name, input shape, output label, train/test split.
    • 
              
  • Model: architecture family and why it matches the data type.
    • 
              
  • Training: epochs, loss, and one changed setting.
    • 
              
  • Testing: one metric table and one visualization.
    • 
              
  • Interpretation: one strength, one failure mode, one limitation.
    • 
              
  • Next step: one experiment that would improve trust in the result.

Source papers and module context

        
References

        

          Each module is connected to a real research workflow. This page helps students separate
          the original scientific contribution from the simplified educational activity in this website.
        


        

          

            Module 1 source paper
            
Wrinkle force microscopy: a machine learning based approach to predict cell mechanics from images

            

              Li, H., Matsunaga, D., Matsui, T. S., Aosaki, H., Kinoshita, G., Inoue, K.,
              Doostmohammadi, A., and Deguchi, S. Communications Biology, 2022.
            

            

              Open PDF
              Open DOI
            

          


          

            
How Module 1 Uses This Paper

            

              

                Original paper
                
Uses microscopy and traction-force data to build WFM, a workflow for predicting cellular force distributions from images.

              

              

                Educational module
                
Focuses on the paired image-to-image learning workflow: input image, target image, generator, discriminator, and visual testing.

              

              

                Important boundary
                
The website demo teaches the machine-learning structure. It should not claim full physical validation of vector force fields.

              

            

          

        


        

          

            
Paper-to-Lesson Map

            

              
                

                  Paper idea
                  Student activity
                

              
              
                

                  Wrinkles are related to cell-generated forces.
                  Inspect paired images and ask what visual information the model can use.
                

                

                  GAN learns from corresponding input and force-distribution images.
                  Run the guided GAN training step and compare generator/discriminator roles.
                

                

                  Predictions must be interpreted carefully in a scientific workflow.
                  Use the result checklist and reflection step before making claims.

    • 
              
  • Which part of the original workflow is simplified in this teaching website?
    •
